<h2>second commit looks like this</h2>
Build a full-stack web-based trading terminal workspace called “Guided Start Mode” for beginners, inspired by modern broker terminals but designed around the philosophy “Out of your way. On your side.” The UI should feel calm, minimal, beginner-safe, and educational instead of overwhelming. Use React + TypeScript + TailwindCSS for frontend, Node.js + Express for backend, PostgreSQL for database, and WebSockets (Socket.io) for real-time market updates, live P&L, and instant order state sync. The layout should include: a beginner progress sidebar (Add Watchlist → Read Chart → Place First Order → Track Position → Unlock Advanced Tools), a simplified candlestick chart with educational tooltips, a watchlist panel, and a 3-step order form (Buy/Sell → Quantity → Plain-English confirmation). Add contextual risk cues like margin impact, daily loss limit, and available funds before confirmation. Include gradual complexity unlocking where advanced features like option chain, market depth, MIS orders, and advanced indicators appear only after user activity milestones. The backend should support JWT authentication, watchlist persistence, portfolio/positions management, order simulation APIs, and WebSocket events for price streaming and order updates. Database schema should include users, watchlists, holdings, orders, alerts, onboarding_progress, and user_preferences tables. Use Redis for WebSocket session caching and rate limiting. Add a “Focus Mode” toggle, auto-save workspace layouts, and a lightweight session journal summarizing trades at the end of the day. The UI should look like a hand-crafted modern trading workspace with clean typography, low visual noise, keyboard shortcuts, responsive panels, and smooth transitions.
